# ⚡ SfaxFix

Plateforme web de services locaux à Sfax (Tunisie).

## 🚀 Fonctionnalités

- Inscription / Connexion utilisateurs
- Gestion des prestataires
- Système de réservation
- Tableau de bord prestataire
- Interface admin
- Système d’avis

## 🛠️ Technologies

- PHP
- MySQL
- HTML / CSS
- JavaScript
