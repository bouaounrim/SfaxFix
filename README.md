# ⚡ SfaxFix

Plateforme web de services locaux à Sfax (Tunisie) permettant aux citoyens de trouver facilement des prestataires et de réserver des services en ligne.

---

## 🚀 Fonctionnalités

* 🔐 Inscription / Connexion sécurisée
* 👨‍🔧 Consultation des prestataires
* 📅 Réservation de services
* 📋 Gestion des réservations (acceptées, refusées, en attente)
* 🛠️ Dashboard prestataire
* 🛡️ Interface administrateur
* ⭐ Système d’avis

## ⚙️ Installation

1. Cloner le projet :

```bash
git clone https://github.com/bouaounrim/SfaxFix.git
```

2. Importer la base de données :

* Fichier : `sfaxfix_db.sql`

3. Lancer avec XAMPP :

* Dossier `htdocs`
* Accéder via :

```
http://localhost/SFAXFIX
```
